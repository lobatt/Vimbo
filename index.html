Vimbo

Weibo client for Vim - inspired by TwitVim

Download as .zip Download as .tar.gz View on GitHub

What is it?

Vimbo is a Weibo client for the VIM text editor, inspired by and based on TwitVim.

What can it do?

For now only user timeline and post to Weibo is supported, more to be done.
